This is a template for Module 04 Activity - Wireframing and Documenting Styling for CIS4655C Section 01 Advanced Mobile Web Application Development - Online Plus - 2019 Winter Quarter Term 2 course. The assignment requires:

Using your own creativity, translate the sketch used in the Module 3 Activity into a wireframe that uses the following style and layout components to the home page or other linked pages:

  • At least 3 buttons that have badges. They can be any size you choose, as well as any style. Check
  • At least 2 labels. Check
  • At least 1 jumbotron. Check
  • At least 1 card. The button on the card must link to another page of content. Check
